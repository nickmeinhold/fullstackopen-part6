# Fullstack Open Part 6 Exercises

This repository contains my solutions for Part 6 of the Fullstack Open course. Each folder corresponds to a set of exercises. Instructions for running each project are provided below.

## Folder Overview

- **unicafe-redux**

  - Exercises: 6.1, 6.2
  - How to run:
    1. Navigate to the folder: `cd unicafe-redux`
    2. Install dependencies: `npm install`
    3. Start the app: `npm start`

- **redux-anecdotes**

  - Exercises: 6.13–6.19
  - How to run:
    1. Navigate to the folder: `cd redux-anecdotes`
    2. Install dependencies: `npm install`
    3. Start the app: `npm start`

- **query-anecdotes** (or `part6` if renamed)
  - Exercises: 6.20–6.23
  - How to run:
    1. Navigate to the folder: `cd query-anecdotes` (or `cd part6`)
    2. Install dependencies: `npm install`
    3. Start the app: `npm start`

## Notes

- Each project uses Vite for development, so `npm start` will launch the local development server.
- Make sure you have Node.js and npm installed.
- For each exercise, refer to the folder's README (if present) for more details.
